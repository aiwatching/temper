Forgot your password?

This server doesn't send emails. To reset your password, contact your administrator — they can issue a reset link from /admin/users → Reset password.

If you are the administrator and you've lost your own password, log into the host running this service and run: 

DATABASE_URL=... \
python -m memory_service.reset_password your-email@example.com

That prints a one-time URL you can use to set a new password.
