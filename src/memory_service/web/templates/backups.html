Full backups

Whole-service snapshots — a Postgres dump (users, keys, blocks, documents, episode metadata, memory snapshots) plus FalkorDB's graph RDB (entities + facts + episode content). super_admin only. Restore is a CLI op (./deploy.sh restore <id>) — it overwrites the live database, so it deliberately isn't a web button.

For per-user, point-in-time memory restore see Snapshots. A daily host timer also runs backups automatically; this page triggers an extra one on demand.
