Pull yesterday's activity from memory + (when MCP is wired) GitLab
and Mantis. Format as a standup update:

  **Yesterday**
  - <bullet>

  **Today**
  - <bullet>

  **Blockers**
  - <bullet, or "none">

Be terse. Real bullets, no filler ("worked on stuff"). If there's
nothing for a section, leave the bullet as `(nothing)`.
