# Temper v0.1.1

Released: 2026-04-08

Initial release

### Features
- Code Graph: tree-sitter Java AST analysis with incremental updates
- Module Registry: define module boundaries with glob patterns, auto-suggest on init
- Knowledge Store: SQLite with causal chains, experiences, full temporal history
- 17 MCP Tools: search_code, get_module, remember, recall, find_causal_chain, etc.
- Semantic Search: external embedding API + cosine similarity
- Interface Map: REST endpoint + public method extraction
- HTML Dashboard: temper export for visualization
- On-demand graph refresh via git status
- Proactive constraint injection in get_file_context

**Full Changelog**: https://github.com/AiON0/temper/compare/main...v0.1.1
